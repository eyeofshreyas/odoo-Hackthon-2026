# Security Policy

## Supported Version

This repository contains the latest hackathon version of TransitOps.

| Version | Supported |
|---------|-----------|
| Latest | ✅ Yes |

---

## Reporting Security Issues

If you discover a security vulnerability, please notify the project team privately instead of creating a public issue.

Please include:

- Description of the vulnerability
- Steps to reproduce
- Possible impact
- Suggested mitigation (if known)

The team will review and address the issue as soon as possible.

---

## Security Practices

TransitOps follows these security principles:

- Secure user authentication
- Role-Based Access Control (RBAC)
- Input validation
- Protection against unauthorized access
- Secure handling of application data
- Regular dependency updates
