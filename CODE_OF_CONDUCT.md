# Code of Conduct

## Our Commitment

We are committed to creating a collaborative, respectful, and inclusive environment for everyone contributing to the TransitOps project.

## Expected Behavior

All contributors are expected to:

- Treat everyone with respect and professionalism.
- Communicate politely and constructively.
- Accept feedback gracefully.
- Collaborate openly with team members.
- Write clean, maintainable, and well-documented code.
- Report bugs and issues responsibly.

## Unacceptable Behavior

The following behavior is not acceptable:

- Harassment or discrimination of any kind.
- Offensive or abusive language.
- Personal attacks.
- Plagiarism or submitting someone else's work as your own.
- Sharing confidential project information without permission.

## Enforcement

Project maintainers reserve the right to remove contributions or restrict participation that violates this Code of Conduct.
